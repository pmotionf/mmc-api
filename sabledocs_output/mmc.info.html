message Track (Nested in mmc.info.Request )

Request track status from the server. At least, either driver, axis, or carrier flag should be on to retrieve the desired information. If the filter is driver range or axis range, it will retrieve any set flag information from the track based on the specified range. If the filter is based on carrier id, it will retrieve any set flag information from the track based on the location of the carrier.

Expected response: mmc.Response.body.info.body.track

Field Type Description
1 line uint32

Line ID.
2 driver bool

Enable retrieving driver information from the server.
3 axis bool

Enable retrieving axis information from the server
4 carrier bool

Enable retrieving carrier information from the server
oneof filter
5 driver_range Range

Retrieve the information from a driver range
6 axis_range Range

Retrieve the information from an axis range
7 carriers mmc.info.Request.Track.Ids

Retrieve the information based on the location of carrier(s)
